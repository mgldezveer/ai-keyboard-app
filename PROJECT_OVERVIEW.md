# AI Keyboard - Project Overview

## Project Summary

AI Keyboard is a fully-featured Android keyboard application with integrated ChatGPT/AI capabilities. Based on comprehensive analysis of the AIType AI Keyboard app screenshots, this implementation replicates all core features using modern Android development practices.

## Quick Start

```bash
cd ai-keyboard-app
./setup.sh
```

Then open in Android Studio and run on device.

## API Integration

### OpenAI Request/Response

```kotlin
// Request
OpenAIRequest(
    model = "gpt-3.5-turbo",
    messages = [
        ChatMessage(role = "user", content = "..."),
        ChatMessage(role = "assistant", content = "...")
    ],
    max_tokens = 150
)

// Response
OpenAIResponse(
    id = "chatcmpl-xxx",
    choices = [Choice(message = ChatMessage(...))]
)
```
